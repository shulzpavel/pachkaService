# Alerting & Monitoring

Документация по мониторингу и алертам вынесена в отдельные файлы для удобства:

- `docs/overview.md` — общий гайд: цепочка, порты, жизненный цикл алерта, тестовые команды.
- `docs/prometheus.md` — сбор метрик, правила, примеры запросов.
- `docs/alertmanager.md` — маршрутизация, группировка, тесты доставки.
- `docs/notifier.md` — обработка алертов и формат сообщений.
- `docs/grafana.md` — provisioning, дашборды, datasource.
- `docs/runbooks.md` — типовые инциденты и действия.

Историю изменений правил/параметров ведём в `docs/CHANGELOG.md`.
